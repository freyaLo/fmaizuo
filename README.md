# fmaizuo
> 技术栈 vue + vue-router + axios + webpack + sass  

# 使用说明
```
# 克隆项目
git clone git@github.com:freyaLo/fmaizuo.git

# 安装依赖(推荐使用cnpm)
npm install

# 本地开发环境 访问http://localhost:8080
npm run dev
```
# 项目说明
> 目前暂时只做到了首页和详情页面，访问maizuowangAPI获取数据，渲染页面，用vuex实现状态管理，还有很多待写页面
